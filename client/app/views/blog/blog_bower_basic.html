Basic usage of bower

What is Bower

It fetches packages for you and places them in the bower_components directory.

Installation

Install bower globally.
npm install -g bower

Bower.json

Place bower.json file in the root folder of the client app. Example of bower.json file below.
{ { "name": "portfolio", "version": "0.0.0", "dependencies": { "angular": "^1.3.0", "json3": "^3.3.0", "es5-shim": "^4.0.0", "bootstrap": "^3.2.0", "angular-animate": "^1.3.0", "angular-cookies": "^1.3.0", "angular-resource": "^1.3.0", "angular-route": "^1.3.0", "angular-sanitize": "^1.3.0", "angular-touch": "^1.3.0", "angular-bootstrap": "latest", "angular-translate": "~2.5.2", "zeroclipboard": "~2.2.0", "angular-zeroclipboard": "~0.4.1", "octicons": "~2.2.0" }, "devDependencies": { "angular-mocks": "~1.3.0", "angular-scenario": "~1.3.0", "userapp-angular": "~2.0.4" }, "appPath": "app", "resolutions": { "angular": "1.3.7" } }

How to install components

In the root folder of the client app run the command below.
bower install
Bower fetchs and installs the components listed into bower.json for you.
