Angular directives

What is Angular directive

You can extend HTML and make easily own reusable components with them.

Example: Repeat elements directive

Goal: creating a component for repeating thumbnail elements on the page.

Let's make angular directive called "thumbnailRepeater". Directive gets its template from the file template/thumbnail_repeater.html.

angular.module('app') .directive('thumbnailRepeater', function () { return { templateUrl: 'template/thumbnail_repeater.html', restrict: 'E' }; });

Template

In the template we go through the model cars and print out the car image, title and text.

{{ car.title }}

{{ car.text }}

Usage

Just place the thumbnail-repeater tag into view.

			
			
			< thumbnail-repeater/>
