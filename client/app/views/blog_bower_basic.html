Basic usage of bower

Installation

Install bower globally.
npm install -g bower

Bower.json

Place bower.json file in the root folder of the client app. Example of bower.json file below. 
			
			
			{
			  {
			  "name": "portfolio",
			  "version": "0.0.0",

How to install components

In the root folder of the client app run the command below.
bower install
Bower fetchs and installs the components listed into bower.json for you.
