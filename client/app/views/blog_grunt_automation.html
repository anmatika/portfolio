Javascript automation with Grunt

With help of grunt you can automatize tasks etc. build

Prequisities

Load grunt tasks is handy tool for automatically tool loading. When using the tool all the components in node_modules are loaded automatically for grunt.

npm install load-grunt-tasks --save-dev

Gruntfile.js

The example Gruntfile below. The following tasks included.

  • Automatic css to sass conversion
  • Wiredep for injecting bower components into index.html
  • Jshint

		
				
		module.exports = function(grunt) {
		  require('load-grunt-tasks')(grunt);
		  
		  grunt.initConfig({
		    
		    // watch automatically sass changes
		    compass: {      
		      dev: { 
		        options: {
		          sassDir: 'sass',
		          cssDir: 'css',
		          watch: true
		        }
		      }
		    },
		    // automatically inject bower components into the app
		    wiredep: {
		      app: {
		        src: ['index.html'],
		        ignorePath:  /\.\.\//
		      },
		      sass: {
		        src: ['sass/{,*/}*.{scss,sass}'],
		        ignorePath: /(\.\.\/){1,2}bower_components\//
		      }
		    },
		    jshint: {
		       all: ['Gruntfile.js', 'js/**/*.js']
		    }
		  });
		  grunt.registerTask('default', ['wiredep', 'jshint', 'compass']);
		};

Usage

Run the default task

grunt
