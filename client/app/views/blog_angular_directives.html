Angular directives

What is angular directive

You can extend HTML and make easily own reusable components with them.

Example: Repeat elements directive

Goal: creating a component for repeating thumbnail elements on the page.

Let's make angular directive called "thumbnailRepeater". Directive gets its template from the file template/thumbnail_repeater.html. 

			
			
			angular.module('app')
			  .directive('thumbnailRepeater', function () {
			    return {
			      templateUrl: 'template/thumbnail_repeater.html',
			      restrict: 'E'
			    };
			  });

Template

In the template we go through the model cars and print out the car image, title and text. It will render a button styled link which routes to given route. 
			
			
			<div ng-repeat="car in cars">
				<div class="col-sm-6 col-md-4">
					<div class="car">
						<img src="{{ car.image }}" alt="...">
						<div class="caption">
							<h3>{{ car.title }}</h3>
							<p>{{ car.text }}</p>
							<p>
								<a href="#{{car.route}}" class="btn btn-primary" role="button">Read</a>
							</p>
						</div>
					</div>
				</div>
			</div>

Usage

Just place the thumbnail-repeater tag into view.

			
			
			< thumbnail-repeater/>
