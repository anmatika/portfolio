What is directive
  • Can make html templates

Example: Repeat elements directive 

		
		
		angular.module('app')
		  .directive('thumbnailRepeater', function () {
		    return {
		      templateUrl: 'template/thumbnail_repeater.html',
		      restrict: 'E'
		    };
		  });

Template

		
		
		<div ng-repeat="car in cars">
			<div class="col-sm-6 col-md-4">
				<div class="car">
					<img src="{{ car.image }}" alt="...">
					<div class="caption">
						<h3>{{ car.title }}</h3>
						<p>{{ car.text }}</p>
						<p>
							<a href="#{{car.route}}" class="btn btn-primary" role="button">Read</a>
						</p>
					</div>
				</div>
			</div>
		</div>

Usage

		
		
		< thumbnail-repeater/>
